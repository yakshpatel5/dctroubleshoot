# dctroubleshoot
A high-performance, single-page web application for Data Center technicians. Streamlines hardware diagnostics, fault code resolution (Mariner/Venera systems), and standardizes L1/L2 troubleshooting workflows with integrated incident logging.
Built to reduce MTTR (Mean Time to Repair) by putting critical CLI commands and hardware fault documentation into a single, high-contrast interface optimized for the data center floor
